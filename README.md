# Pro-Gov

## Προαπαιτούμενα
Βεβαιώσου ότι έχεις εγκατεστημένα τα εξής:
 -**JAVA JDK**
 -**Maven**

Για να γίνει ο έλεγχος αυτός, τρέξε:
```bash
java -version
mvn -version
```

## Οδηγίες μεταγλώττισης του προγράμματος

Η εφαρμογή έχει υλοποιηθεί ως έργο Maven. Για τη μεταγλώττιση απαιτείται η εγκατάσταση του Java Development Kit (JDK) και του Maven στο σύστημα.

Από τον βασικό φάκελο του αποθετηρίου εκτελείται η παρακάτω εντολή:

```bash
mvn clean package
```

## Οδηγίες χρήσης του προγράμματος

Η εφαρμογή **Pro-Gov** είναι πρόγραμμα γραμμής εντολών (console) για την προβολή και διαχείριση στοιχείων κρατικού προϋπολογισμού.

### Εκκίνηση προγράμματος

Η εκτέλεση της εφαρμογής γίνεται από το τερματικό με την παρακάτω εντολή:

```bash
java -jar ProGov.jar
```

## Παρουσίαση της δομής των περιεχομένων του αποθετηρίου

Το αποθετήριο του έργου ακολουθεί τη δομή ενός τυπικού έργου Maven και είναι οργανωμένο με τρόπο που διαχωρίζει τη λογική της εφαρμογής, τα δεδομένα και τη διεπαφή χρήστη.

### Φάκελοι

- **src/main/java**: Περιέχει τον κύριο πηγαίο κώδικα της εφαρμογής.
- **src/test/java**: Περιέχει αρχεία δοκιμών (tests) για τον έλεγχο της λειτουργικότητας της εφαρμογής.

### Package com.progov

Το βασικό package της εφαρμογής είναι το `com.progov`, το οποίο περιλαμβάνει τα παρακάτω υπο-packages και αρχεία:

- **app/**: Περιέχει κλάσεις που σχετίζονται με τη βασική ροή και εκκίνηση της εφαρμογής.
- **model/**: Περιέχει κλάσεις που αναπαριστούν τα δεδομένα του προϋπολογισμού και τις βασικές οντότητες της εφαρμογής.
- **service/**: Περιέχει κλάσεις που υλοποιούν τη λογική επεξεργασίας των δεδομένων.
- **ui/**: Περιέχει κλάσεις που σχετίζονται με τη διεπαφή χρήστη (μενού, αλληλεπίδραση).

### Κύρια αρχεία Java

- **Users.java**: Διαχειρίζεται τους ρόλους χρηστών (π.χ. Πολίτης, Υπουργός) και τις αντίστοιχες επιλογές.
- **MenuEpilogwn.java**: Υλοποιεί τα μενού επιλογών της εφαρμογής.
- **Epiloges.java**: Περιέχει λειτουργίες που σχετίζονται με την προβολή και τροποποίηση του προϋπολογισμού.
- **Trial.java**: Αρχείο δοκιμών και πειραματισμού λειτουργιών.
- **CheckVariables.java**: Περιλαμβάνει ελέγχους εγκυρότητας εισόδου χρήστη.
- **BudgetComparator.java**: Υλοποιεί λειτουργίες σύγκρισης προϋπολογισμών.
- **CompareCharts.java**: Αφορά την οπτική σύγκριση δεδομένων μέσω γραφημάτων.
- **Diagram.java**: Σχετίζεται με τη δημιουργία ή απεικόνιση διαγραμμάτων.
- **DatabaseConnection.java**: Διαχειρίζεται τη σύνδεση της εφαρμογής με τη βάση δεδομένων.

Η παραπάνω δομή επιτρέπει την ευκολότερη συντήρηση και επέκταση της εφαρμογής, διαχωρίζοντας καθαρά τις αρμοδιότητες κάθε τμήματος.

## Επισκόπηση δομών δεδομένων και αλγορίθμων

Η εφαρμογή Pro-Gov βασίζεται σε απλές και αποδοτικές δομές δεδομένων της Java, καθώς και σε βασικούς αλγορίθμους επεξεργασίας, ώστε να διαχειρίζεται και να αναλύει στοιχεία κρατικού προϋπολογισμού.

### Δομές Δεδομένων

- **Πίνακες (arrays)**  
  Χρησιμοποιούνται μονοδιάστατοι πίνακες (`int[]`, `String[]`) για την αποθήκευση:
  - κατηγοριών εσόδων και εξόδων
  - αντίστοιχων αριθμητικών τιμών (ποσά προϋπολογισμού)

  Παράδειγμα:
  - `String[] categories`
  - `int[] values`

- **Μεταβλητές ακεραίων (int)**  
  Χρησιμοποιούνται για:
  - αποθήκευση χρηματικών ποσών
  - αθροίσματα εσόδων και εξόδων
  - επιλογές χρηστών από τα μενού

- **Μεταβλητές λογικού τύπου (boolean)**  
  Χρησιμοποιούνται για έλεγχο εγκυρότητας εισόδου και για επαναλήψεις με συνθήκη.

---

### Αλγόριθμοι και Λογική Επεξεργασίας

- **Επαναληπτικές δομές (for, do-while)**  
  Χρησιμοποιούνται για:
  - διάσχιση πινάκων
  - υπολογισμό συνολικών εσόδων και εξόδων
  - επαναλαμβανόμενη εμφάνιση μενού μέχρι την έξοδο του χρήστη

- **Δομές επιλογής (if-else, switch)**  
  Χρησιμοποιούνται για:
  - έλεγχο εγκυρότητας εισόδου χρήστη
  - επιλογή λειτουργιών από τα μενού
  - σύγκριση εσόδων και εξόδων (πλεόνασμα / έλλειμμα)

- **Αλγόριθμος αθροίσματος**  
  Τα συνολικά έσοδα και έξοδα υπολογίζονται με επαναληπτική πρόσθεση των στοιχείων των πινάκων.

- **Αλγόριθμος σύγκρισης**  
  Συγκρίνονται τα συνολικά έσοδα και έξοδα ώστε να εξαχθεί συμπέρασμα για την οικονομική κατάσταση (πλεονασματικός, ελλειμματικός ή ισοσκελισμένος προϋπολογισμός).

---

Η χρήση απλών δομών δεδομένων και βασικών αλγορίθμων καθιστά την εφαρμογή εύκολα κατανοητή, επεκτάσιμη και κατάλληλη για εκπαιδευτικούς σκοπούς.

## Έλεγχοι (Testing)

Πραγματοποιήθηκαν βασικοί έλεγχοι λειτουργικότητας με JUnit, εστιάζοντας στην ορθότητα των υπολογισμών του κρατικού προϋπολογισμού, όπως τα σύνολα εσόδων και εξόδων, καθώς και στο τελικό ισοζύγιο. Οι έλεγχοι εκτελέστηκαν με την εντολή:

```bash
mvn test
```
